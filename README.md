# ZTweaks-Free-Macros
The ZTweaks keyboard macro is overpriced and I don't think anyone should give them money. I managed to copy the AHK scripts for the macros so anyone can use them for free.

The devs obviously didn't want anyone to do this, so they made the script files only generate while the macro is running. However, it was really easy to copy them while the macros were running and paste them elsewhere.
```
To use the macros, click on whichever file you want to use and it will automatically turn on the macro. You can turn it off by going to the tray and closing it.
To change the keybinds, right click on the macro you want to edit and open it in VSCode. Then find the global variables section and change them to whatever you want.
You can also change the timing and even add functions to this macros this way.
```
And no, this isn't illegal. By a stroke of genius, the ZTweaks owners licensed all their products under the Apache License, which allows for free distribution and modification. The ZTweaks owners can't sue me or take down this repo.

Enjoy!
